# OnlineShopProject-JavaFX

A simple online shop desktop application built with **Java** and **JavaFX** for an advanced programming class. The repository is structured as a Maven-based Java project and includes a modular setup (`module-info.java`) together with JavaFX UI screens for the main shopping flow.

## Features

- JavaFX desktop UI for an online shopping experience.
- Login and signup pages.
- Main menu and customer menu screens.
- Product page and review/score pages.
- Customer credentials management and add-fund flow.
- Shopping cart and product-related images/assets included in the UI.

## Project Structure

The project follows a Maven layout and contains:

- `pom.xml`
- `mvnw` / `mvnw.cmd`
- `.mvn/`
- `src/main/java`
- `src/main/resources`
- `module-info.java`

## Technologies Used
